如何用hugo搭建个人博客

如何用hugo搭建个人博客 先去Hugo releases页面下载hugo_xxx_Windows-64bit.zip(目前已经是hugo_0.72.0_Windows-64bit.zip )不想费劲的可以参考芳芳老师的连接直接下载 尽量按教程上的目录配置：D:\Software\hugo\ 目录下解压文件，得到hugo.exe文件 把第二部路径配置到path中，然后重启终端，输入hugo version查看版本 进入Hugo官网，点击按钮Quick start快速搭建，按照步骤进行操作 因为第一步已经下载过所以不需要执行，直接从第二步开始，执行到第7步 会得到一个public目录，这个目录是博客站点（该目录无法双击预览，可以用hugo server预览） 在外层目录中创建,gitignore文件，把public文件放进去 在github中创建仓库仓库名要用github用户名.github.io 进入public文件，初始化本地仓库运行,运行如下代码 git init,git add ., git commit -m "命名" 然后github中复制两段生成的代码 在github中查看setting选项，找到github pages 没有配置过的会有一个按钮，选择master 按钮上一行会生成一个网址，点击就可以查看自己的博客了
